Getting started

OptiRoute
Find the optimal way to deliver goods to your customers!

OptiRoute allows you to solve vehicle routing problems with time windows.

Depot

In this step, specify problem details and set the depot location.
  • Start date - when delivery should start
  • Due date - the final time at which the vehicles should arrive
  • Vehicles - number of vehicles used for delivery
  • Capacity - capacity of each vehicle
Remember to place the depot marker on the map, without this step you will not be able to continue.

Customers

In this step you need to set the location and specify the details of your customers.
Place the client marker on the map and the client details section will appear in the sidebar.
  • Ready time - the earliest date on which the customer can take delivery
  • Due date - final date on which the customer may take delivery
  • Service time - time taken to accept delivery
  • Demand - quantity of goods in delivery

Solution

If the problem is feasible, it will be visualized on the map.
The sidebar provides details of each route in the solution.
